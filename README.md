## Project-IPPIE

- Este projecto esta na faze de desenvolvimento
- Projecto Open Source

clone Projecto

```
git clone https://github.com/efrahim20319/Project-IPPIE.git
```

```
cd Project-IPPIE
npm install
```
or
```
cd Project-IPPIE
yarn
```

start Project
```
npm run start
```
or
```
yarn start
```
